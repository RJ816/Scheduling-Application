Title: Scheduling Application
Purpose: Program to manage customers, appointments, and generate custom reports for an international consulting firm.
Student application version: 1.0
Date: 9/8/23
IDE: IntelliJ IDEA Community Edition 2021.1.3 
JDK: Java SE 17.0.1
JavaFX: JavaFX-SDK-17.0.1
Directions: Login with valid credentials. A customer and appointment directory can be viewed, and both can add, update, and delete their respective lists. Custom reports can be viewed based on available data. 
Additional Report: Sums the total appointments based on location.
MySQL Connector driver: mysql-connector-java-8.0.25
